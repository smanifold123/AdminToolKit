A modern, modular **Windows SysAdmin toolkit** built with **PowerShell**, **WPF**, and **Fluent Ribbon**.

## ⭐ Features

### 🖥️ Windows Administration
- AD user provisioning & offboarding  
- Group Policy quick‑actions  
- Local admin & service management  
- WinRM automation

### ⚙️ Automation & Scripting
- Modular PowerShell backend  
- Extensible plugin architecture  
- Logging & audit trail support  

### 🎨 Modern WPF UI
- Fluent Ribbon interface  
- Dark/light theme  
- Responsive layouts
