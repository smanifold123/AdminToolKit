# Installation Guide

## Requirements
- Windows 10/11  
- PowerShell 5.1 or PowerShell 7  
- .NET Framework (for WPF UI)

## Install
1. Download the latest installer from the Releases page  
2. Run the MSI or EXE  
3. Launch **AdminToolKit** from the Start Menu

## Updating
Simply install the new version — settings are preserved.
